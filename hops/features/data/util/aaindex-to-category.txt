# Map each aaindex feature to a category. These categories are not strictly separable, but
# should provide some information about the underlying features of amino acids that are
# being discovered by the machine learning tool. 
#
# The categories are:
#
# charge        Information about charge state, ionization, pKa values, etc.
# composition   Composition is a rough category holding things like the relative frequency of
#               this amino acid in some environment or some class of proteins.
# flexibility   Measures of flexibility.  These include conformational entropy terms, linker
#               propensity, and actual flexibility parameters.
# geometry      Sidechain geometry parameters including molecular weight, volume, typical 
#               contact order, various reduced descriptions of sidechain geometry, etc.
# hydrophobic   Measures of hydrophobicity including transfer free energies, calculated 
#               solvation energies, fraction buried, and surface area
# meta          Parameters that are, themselves, aggregate PCA-like parameters that cannot be
#               usefully decomposed
# observable    Some observable like refractive index or chemical shift that cannot be easily
#               connected back to a physical property
# polar         Measures of polarity and hydrophilicity.  This could probably be effectively
#               combined with hydrophobicity into a single category. 
# sec_struct    Secondary structure propensity scores, including turns.
# 
#aaindex      category      aaindex description
